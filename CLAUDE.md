# 프로젝트 가이드

이 프로젝트의 작업 규칙은 **스킬(Skill)** 로 관리한다.

세부 규칙을 CLAUDE.md 에 중복 기재하지 않고, 아래 스킬 목록을 통해 참고한다.

새 규칙이 생기면 `.claude/skills/<skill-name>/SKILL.md` 로 스킬을 추가하고, 이 문서의 목록에 한 줄 링크를 더한다.
